What is a process??

A process is a program that has its own process ID (PID) and
memory space, so it doesn't compete for resources with other 
running processes.

You can clone a process using the
fork() system call, which creates a separate memory space
for this clone.  The clone is called the child process and 
the original is called the parent.

You can wait for the child to terminate inside the parent
using functions such as wait() and waitpid().  Which can
also be used to obtain some information about the child process
such as its exit status.
